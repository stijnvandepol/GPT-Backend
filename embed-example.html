Mijn Portfolio

Dit is een voorbeeld van hoe je de chatbot widget kan integreren in je portfolio website.

Klik op de chat bubble rechtsonder om de assistent te openen!

Hoe te integreren:

<!-- Voeg dit toe aan het einde van je <body> tag -->
<script>
  window.CHATBOT_CONFIG = {
    apiUrl: 'http://localhost:3000' // Vervang met jouw server URL
  };
</script>
<script src="http://localhost:3000/chatbot-widget.js"></script>

Voor productie:

<script>
  window.CHATBOT_CONFIG = {
    apiUrl: 'https://jouw-api-server.com'
  };
</script>
<script src="https://jouw-api-server.com/chatbot-widget.js"></script>
